# CalmSphere

Este proyecto es una landing page para la aplicación CalmSphere, diseñada para ayudar a personas con estrés y ansiedad. Fue desarrollado utilizando HTML, CSS y JavaScript.

## Características

- Diseño responsivo.
- Interfaz intuitiva.
- Animaciones con JavaScript.
